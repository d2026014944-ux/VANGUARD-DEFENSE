# VG-SKILL-MANAGER

Sistema de gerenciamento das skills do projeto.

## Estrutura esperada

- `skills/<skill-id>/SKILL.md`

## Skills registradas

### Core Toolkit
- docx
- xlsx
- pdf
- pdf-reading
- pptx
- frontend-design
- file-reading

### Power-User
- skill-creator
- mcp-builder
- web-artifacts-builder

## Objetivo

Garantir governança: o registro de skills e os arquivos `SKILL.md` obrigatórios
devem existir e permanecer sincronizados.
