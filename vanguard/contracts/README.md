# Contratos de Interface (Schema-First)

- `detection.schema.json`: contrato do payload de detecção produzido pelo VG-VISION.
- `cot-event.xsd`: contrato XML-CoT mínimo exigido entre VG-CORE e VG-COMM.

Os contratos são versionados e devem ser atualizados antes de mudanças incompatíveis na implementação.
